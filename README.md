# Green Flag Certification Analytics — Python & Power BI

**MSc Group Project | Maynooth University | Business Analytics 2024–2025**

---

## 📌 Project Overview

Data analysis of Irish school energy and sustainability data to support Green Flag certification through the Green-Schools Programme (An Taisce). Raw national datasets were collected, cleaned using Python and R, and visualised through interactive Power BI dashboards.

---

## 📁 Repository Structure

### ORIGINAL DATA

Raw datasets collected from:

- SEAI (Sustainable Energy Authority of Ireland) — seai.ie
- Kaggle — Global Data on Sustainable Energy
- Kaggle — Green Energy Dataset
- Kaggle — Electricity Generated by Fuel Type
- Ireland–Germany CO₂ Comparison dataset
- Insulation Level and CO₂ Emissions dataset

### CODE DOWNLOADED

Python and R scripts used to clean and analyse the original data:

- Data cleaning and standardisation using **Pandas**
- Exploratory analysis and visualisations using **Matplotlib**
- **ANOVA** statistical analysis of insulation levels vs energy demand
- **ANOVA** statistical analysis of CO₂ emission sources
- Both ANOVA tests returned p-values below 0.05 — statistically significant results

### CLEANED

Cleaned and processed datasets produced by running the Python and R scripts on the original data. Ready for analysis and dashboard development.

### POWER BI VISUALISATION

Four interactive Power BI dashboards built using the cleaned datasets:

- Ireland vs Germany CO₂ Comparison
- Annual Energy Statement trends
- Electricity Generated by Fuel Type
- Green Energy Dataset — insulation and payback analysis
